PLTW 1.1.9 Project

Build Your Own Snowman

The program uses if statements to decide what should be put on the snowman. For instance, you can decide whether or not it should have a hat or a scarf. The program also asks if you want to build a snowman in the beginning and if you state 'no' then the program exits.

PLTW 1.2.5 Project

Whack-a-Minion

The program has a list of coordinates which are the coordinates of the holes. The program uses the random module in order to select a random point for the minion to pop up from. When the minion is clicked, it goes to another random point and the loop is continued. The loop stops when the timers runs out.

PLTW 1.3.1 Project

Snowman Joy

This program is an interactive e-card. The program will ask for your name before it will run. After that, a card will appear and you will need to click the wax to open it. After it opens, it sets the background, plays music, draws a snowman, writes a message and then generates snow that iterates.

PLTW 2.1.6 Project

Fish Tank Monitoring

This project focused on debugging code from a suspicious dysfunctional fish tank monitoring software. After using many debugging methods, I was able to fix the code resulting in a functional fish tank monitoring program.

Scratch Project

Flappy Bird

This project was a recreation of Flappy Bird in Scratch. It includes different levels that have different backgrounds and randomly generated obstacles. The different levels are achieved by getting a higher score. Here's the link to the game: https://scratch.mit.edu/projects/1096085222.
